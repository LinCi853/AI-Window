# AI 窗口

一个窗口装下 9 个 AI，按 Alt+1~9 秒切，对比答案不用再来回切浏览器。

## 有什么用

| 场景 | 怎么做 |
|------|--------|
| 问一个问题，看 3 个 AI 的不同答案 | Alt+1 切 DeepSeek、Alt+3 切 ChatGPT、Alt+6 切 Kimi，秒切不丢会话 |
| 不想打字，直接说话问 AI | 按住 Alt+V 说话，松开自动转文字发出去，窗口都不用开 |
| 边写文档边问 AI | 按 F12 置顶，AI 窗口浮在最前面，切到 Word 也挡不住 |
| 所有对话自动存本地 | 不用手动保存，随时搜关键字找回 |

---

## 怎么用

**最少只要会 `Alt+Space` 呼出窗口和打字，就能用。**

1. 下载安装后启动，窗口窄长贴边，不挡你正在做的事
2. 输入框已聚焦好，直接打字提问
3. 按 `Alt+1`~`9` 切换不同 AI，每个 AI 的会话独立保留

## 快捷键速查

| 快捷键 | 干什么 | 哪能用 |
|--------|--------|--------|
| `Alt+Space` | 呼出 / 隐藏主窗口 | 任何地方 |
| `Alt+Q` | 显隐独立 AI 窗口 | 任何地方 |
| `Alt+V`（按住） | 说话转文字发给 AI | 任何地方 |
| `Alt+1`~`9` | 切到第 N 个 AI | 窗口内 |
| `Ctrl+Tab` | 切到下一个 AI | 窗口内 |
| `Ctrl+T` / `Ctrl+W` | 新建 / 关闭标签 | 窗口内 |
| `F4` / `F6` | 后退 / 前进 | 窗口内 |
| `F5` | 刷新当前页 | 窗口内 |
| `F10` | 切换深色 / 浅色 | 窗口内 |
| `F11` | 窗口最大化 / 还原 | 窗口内 |
| `F12` | 窗口置顶（最大化时不可用） | 窗口内 |

## 内置 AI 平台

DeepSeek、豆包、Kimi、ChatGPT、Claude、Gemini、Perplexity、通义千问、智谱清言。

也支持添加任意自定义网页 URL。
